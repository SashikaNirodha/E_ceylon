Application name : E Ceylon

Description :

E ceylon is an mobile application that designed to simplify the process of pay bus tickets. 
This system allows users to make secure payments online. It also provides administrative 
features for bus owners to view ticket processing process and money management efficiently. 
The goal of the project is to reduce manual effort, and enhance the overall ticket payment 
experience through digital convenience. Our project aims to develop a mobile application that 
facilitates seamless payments in public transport (specifically buses) through a digital wallet 
system. The motivation behind this project is to modernize the payment infrastructure of public 
transport, providing a convenient and secure alternative to cash. This application will 
significantly benefit the industry by streamlining payment collection for bus owners, offering a 
hassle-free experience for passengers, and providing a more transparent money change tracking 
system for all stakeholders. 

Aim and Objectives : 

The aim of this project is to develop a comprehensive mobile payment application specifically 
designed for public transport, enabling seamless online transactions for passengers, conductors, 
and bus owners. This application intends to modernize the traditional cash-based payment 
system by providing a digital platform that simplifies and secures the payment process. Through 
this system, passengers will be able to make hassle-free payments, conductors can easily 
manage fare collections via QR code generation. Additionally, the project will integrate a 
money agent dashboard to facilitate the exchange of physical money into virtual currency, thus 
creating an interconnected and efficient digital payment ecosystem for the entire public bus 
transport network.  
The objectives of the project are as follows: 
• To develop a passenger dashboard for transfer money . 
• To develop a conductor dashboard for receive money . 
• To develop an owner dashboard to facilitate see money transfer procedure. 
• To develop a money agent dashboard for exchange physical money to virtual money.
